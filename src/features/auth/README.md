# Auth Feature

This directory contains all logic related to authentication.

## Structure

- `api/`: API request functions (e.g., `login`, `register`).
- `components/`: UI components specific to auth (e.g., `LoginForm`).
- `hooks/`: Custom hooks (e.g., `useAuth`).
- `types/`: TypeScript interfaces and types.
- `routes/`: Route definitions for this feature.
